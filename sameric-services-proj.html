Project Brief

Sameric Services is a website for a property management company that works with local companies and businesses and specialises in the housing of working proffesionals within there respective area. The initial website was created using a generic wix template and was not responsive meaning it would not show well on smaller devices it was also quite dull and overall presentation was not great. veiw old website. The aim is to create a new and improved responsive website that is unique and stands out all whilst getting the companies message across.

Outcome

I used mainly HTML CSS and Jquery to create a much better website from scratch that utilises the mobile fist appoach to make sure it still looks good on smaller devices. I also used slick carousel in order to make a much better way to show images of the properties under mangement at first I ran into a few issues making my code run however I stayed up all night in order to solve the issue and realised I did not have charset="UTF-8" in my head section which is why the images where not initially showing. I also cut out alot of uneccessary text and made the sections of the website alot clearer and thus easier for clients to understand key points all in all i feel it is much better.
