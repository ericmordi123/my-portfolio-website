Project Brief

Using known Javascript libraries Create a site that calls on the Google Maps API and/or the Google Places API (or similar) to allow users to search for their next holiday destination.

Outcome

Utilising the google maps and google places API I have created a single webpage application that shows you the best to hotels to saty in, restuarts to eat at and places to visist anywhwere in the UK.
